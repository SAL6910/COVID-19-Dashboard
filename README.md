# COVID-19-Dashboard

## About the Project

This project is a COVID-19 Dashboard created using HTML and D3.js v7. The dashboard makes use of the COVID-19 dataset: https://github.com/owid/covid-19-data/blob/master/public/data/owid-covid-data.csv
